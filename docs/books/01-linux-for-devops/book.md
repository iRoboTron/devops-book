## Как пользоваться книгой

1. **Читай по порядку** — главы построены друг на друге
2. **Делай упражнения** — теория без практики мертва
3. **Отмечай чеклисты** — в конце каждой главы
4. **Не пропускай** — если что-то непонятно, перечитай
5. **Практикуйся на виртуалке** — ломай, чини, учись

## Предварительные требования

- Установленная Ubuntu Server (физическая или виртуальная)
- Доступ к терминалу
- Желание учиться
