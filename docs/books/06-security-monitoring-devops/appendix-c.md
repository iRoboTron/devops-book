# Приложение C: Признаки взлома

> Если подозреваешь компрометацию — проверь по этому чеклисту.

---

## C.1 Новые пользователи

```bash
getent passwd | grep -v nologin | grep -v false
```

Подозрительно: пользователи которых ты не создавал.

---

## C.2 Последние входы

```bash
last | head -20
```

Подозрительно: входы в странное время, с незнакомых IP.

---

## C.3 Подозрительные процессы

```bash
# Топ по CPU
ps aux | sort -rk 3 | head -10

# Топ по памяти
ps aux | sort -rk 4 | head -10
```

Подозрительно: `xmrig`, `kdevtmpfsi`, `minerd` — майнеры.
Подозрительно: процессы с рандомными именами.

---

## C.4 Исходящие соединения

```bash
ss -tnp | grep ESTABLISHED
```

Подозрительно: соединения на странные порты (не 80, 443, 53).
Подозрительно: соединения на известные C&C серверы.

---

## C.5 Изменённые файлы

```bash
# Файлы изменённые за последние 24 часа
find /usr /bin /sbin -mtime -1 -type f 2>/dev/null

# SUID файлы
find / -perm /4000 -type f 2>/dev/null
```

Подозрительно: новые SUID файлы.
Подозрительно: изменённые системные бинарники.

---

## C.6 Cron всех пользователей

```bash
for user in $(getent passwd | cut -d: -f1); do
    crontab -u "$user" -l 2>/dev/null | grep -v "^#" | grep -v "^$"
done
```

Подозрительно: cron задачи которых ты не создавал.
Подозрительно: скрипты скачивающие что-то из интернета.

---

## C.7 Autorun

```bash
# systemd сервисы
systemctl list-unit-files --state=enabled | grep -v "^$\|static\|indirect"

# rc.local
cat /etc/rc.local 2>/dev/null
```

Подозрительно: неизвестные сервисы.
