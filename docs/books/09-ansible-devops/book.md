## Главная идея

Ansible решает: **"одинаковая конфигурация на всех серверах"**.

```
Shell-скрипт:           Ansible:
./setup.sh              ansible-playbook setup.yml
↓                       ↓
последовательно         параллельно
не идемпотентно         идемпотентно
```

Идемпотентность: запусти дважды — результат одинаковый.

## Сквозной проект

Через всю книгу проходит один проект: **деплой Flask-приложения за Nginx**.

```text
Глава 1  → описали сервер в inventory
Глава 4  → поставили Nginx и создали deploy user
Глава 5  → сгенерировали nginx-конфиг из шаблона
Глава 6  → вынесли всё в роли
Глава 7  → спрятали секреты
Глава 9  → доказали идемпотентность и добавили тесты
```
